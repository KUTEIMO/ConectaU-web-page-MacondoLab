# APK de Android

Coloca aquí el archivo `app.apk` de tu aplicación móvil de Android.

El archivo debe llamarse exactamente `app.apk` para que el enlace de descarga funcione correctamente.

## Instrucciones:

1. Genera tu APK de Android
2. Renómbralo a `app.apk`
3. Colócalo en esta carpeta (`public/app.apk`)

La aplicación web buscará el archivo en la ruta `/app.apk` cuando los usuarios hagan clic en el botón de descarga.
